## How to Deploy on GitHub Pages

1. Create a new GitHub repository, for example `portfolio`.
2. Upload all files from this folder to the repository.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save and wait for GitHub to publish the site.

Your website URL will look like:

```text
https://your-github-username.github.io/portfolio/
```

## Customize Before Publishing

Update these items in `index.html`:

- GitHub profile link
- LinkedIn profile link
- Behance link
- Real project links
- Phone or WhatsApp number
- Download CV link, if you add a PDF resume

## Notes

The contact button uses a `mailto:` link, so it works without a backend server. If you want a real contact form later, connect it with Formspree, Netlify Forms, or your own backend API.
